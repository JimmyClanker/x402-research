API / MCP

Plug in from anywhere

Simple REST or native MCP protocol. Built for AI agents, developers, and anyone who wants programmatic crypto research.

# Streamable MCP (Model Context Protocol)
POST https://api.clawnkers.com/mcp

# crypto_research tool
{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "crypto_research",
    "arguments": { "query": "solana" }
  }
}

# url_extract tool
{
  "jsonrpc": "2.0",
  "method": "tools/call",
  "params": {
    "name": "url_extract",
    "arguments": { "url": "https://example.com" }
  }
}

# Auth: x402 USDC payment on Base or API key
# x-api-key: YOUR_KEY

# Example response
{
  "verdict": "BUY",
  "project_name": "Solana",
  "scores": {
    "market_strength": { "score": 8.5 },
    "onchain_health":  { "score": 7.8 }
  },
  "llm_analysis": {
    "analysis_text": "Strong momentum…",
    "moat": "Speed + ecosystem lock-in",
    "risks": ["Centralization", "Validator concentration"],
    "catalysts": ["ETF inflows", "DePIN growth"]
  }
}
Endpoint

POST https://api.clawnkers.com/mcp — Streamable HTTP transport, compatible with any MCP client.

Tools

crypto_research(query) — Full alpha scan with AI verdict.
url_extract(url) — Clean content extraction from any URL.

Auth & Pricing

Two options:
1. x402 — Send 1.00 USDC on Base per request (no API key needed).
2. API Key — Header x-api-key: YOUR_KEY for bulk/agent use. Contact us for a key.

REST fallback

GET /alpha?project=solana — Full scan (paid).
GET /alpha/quick?project=sol — Free algorithmic scan.
GET /api/health — Server status & scan count.

Error codes

400 — Bad request (missing/invalid params)
402 — Payment required (x402 flow)
429 — Rate limited — slow down
500 — Server error — try again
